Hidden & Under-Used Features

10 — power tools

Almost all of these live behind the right-click context menu or the Expert toolbar (hover the light strip at the bottom edge of the CAD window to slide it open).

Magic Lantern

Repositions the virtual light source in the scene — just drag the pop-up. Purpose-built for reading tricky, shadowed margins where the default lighting hides the finish line. Context menu → Magic Lantern.

Register Meshes

3-point alignment between any mesh objects in the CAD window. Align a pre-op over a prep scan, drop a diagnostic waxup onto the working model, or seat an imported STL reference — without leaving the design session.

3D PDF Export

Exports the current CAD view as a rotatable 3D PDF anyone can open in Adobe Reader. The most underrated doctor-communication tool in the software — no viewer install, no portal login.

Auto Hide

Group Selector option that automatically hides the opposing jaw whenever your view direction faces it. Set it once and stop toggling A forever.

The "Hidden" Expander

Lost an object? Unfold the Hidden group in the Group Selector to see everything currently invisible, or hit Show all to reset visibility in one click.

Expert Toolbar via Selection

Click an object (or Ctrl-click several) and the Expert toolbar populates with only the tools valid for that selection. Faster than digging through the full context menu, and it teaches you what's possible per object type.

Wizard ⇄ Expert Round-Tripping

Suspending the wizard doesn't lose your place — press the Wizard button and it resumes at the exact step you left. Dip into expert mode for one surgical fix, then let the wizard carry on.

TruSmile Toggle

Photorealistic rendering on demand from the main toolbar. Design in flat shading for speed, flip TruSmile on only for esthetic evaluation and screenshots for the doctor.

2D Image Overlay Hacks

The 2D Image Loader (expert mode) accepts any PNG — techs load golden-ratio proportion grids and smile-design guides as semi-transparent overlays for anterior setups. Community-made overlay packs circulate on Dental Lab Network.

Reference Meshes as Tools

The Additional Mesh loader will import any STL into the scene — calibration spheres, custom arch guides, printable jigs — usable as visual references or Boolean partners during design.

Minor Tweaks & Workflow Habits

11 — lab-floor wisdom

Per-Material, Per-Client Defaults

In DentalDB, open Advanced parameters on the right side of the tooth/material selection. Cement gap, milling parameters, and minimum thickness set here are saved as the new default for that material and that client — set once per doctor/mill combo, never touch again.

Re-Open Sealed Designs

Finished case but need to loosen the fit? Load it → Expert → delete the merged parts → jump to Crown Bottoms → adjust gaps → press Wizard to re-merge. No restart, no redesign.

Crown-Bottom Edits Are Per-Case

Gap changes made during the design phase (Crown Bottoms step) apply only to that restoration and won't persist. Permanent changes belong in DentalDB's advanced parameters — know which layer you're editing.

Pivot Before You Sculpt

Habit: MMB-click the unit you're about to work on before starting free-forming or margin work. Every rotation then orbits your working area instead of the model centroid.

Screenshot Trail

Ctrl+Shift+Q at every major step drops timestamped captures into the project folder — a free design-decision audit trail for remakes and doctor disputes.

Settings Menu Depth

The main Settings dialog (gear icon) covers language, tooth numbering scheme (FDI/Universal), and graphics options. If performance drags on older GPUs, the graphics settings here are the first stop before blaming the scan.

Custom Views as Insertion Paths

On cases with multiple paths of insertion, save one named custom view (Ctrl+F1) per path. Ctrl+F2 then cycles your exact evaluation angles for undercut checks.

Ctrl+Q Is the Manual

Can't remember where a tool lives? The free-text search finds functions by name and jumps you to them. Faster than the wiki, faster than the context-menu spelunk.
