Hidden & Under-Used Features

07 — power tools

Shift + F4 — CAM Output Folder

Opens the CAM output folder directly in Explorer from Dental Manager. No more digging through 3Shape/DentalSystem/CAM output paths by hand when the mill software wants a file — this is the shortcut nobody teaches and everybody needs.

Ctrl + W — 3D Preview Without Designer

Full rotatable 3D preview of any designed order straight from the Manager list. QC pass on the day's queue, or show a doctor a design on the phone, without a single Designer launch (and its load time).

Appendix B Is the Real Manual

Press F1 → table of contents → Appendix B: Using Mouse and Keyboard. The complete shortcut list for your exact version including every installed add-on module — the printed lists floating around forums are subsets of this.

Control Panel Import/Export (DME)

Materials, anatomy libraries, implant systems, and manufacturing settings all travel as DME files via Control Panel → Import/Export. Back up your entire lab configuration before every version migration, and use it to clone settings across design stations.

Scan-Your-Own Anatomy Libraries

Any waxup you scan can become a permanent library tooth: outline the cutoff, set the characteristic points (cusp, groove, contact, vestibular), and save. Your best hand-waxed anatomy becomes a starting point for every future case.

Batch Filters Are Selection Tools

The Ctrl+Alt selectors (patient / client / status / date) don't just highlight — the selection feeds batch operations. Select same-delivery-date, then export or advance the whole set at once.

3D Mouse Support

Dental System natively supports 3D mice (SpaceMouse class). Left hand navigates the model continuously while the right hand keeps the pen or mouse on the tools — the setup 3Shape's own power users run.

Graphics Tablet Support

Sculpting with a pen tablet gives pressure-style control over the wax knife that a mouse can't match. Officially supported — worth trying if there's a tablet gathering dust near the render machine.

Minor Tweaks & Workflow Habits

08 — lab-floor wisdom

Build the Preset Kit First

Ten minutes configuring Ctrl+1–6 wax knife presets pays back on every single case afterward. Store your presets as part of onboarding any new design station — they're per-installation, not per-case.

M → Move → N

When an anatomy library tooth fights the margin, disconnect (M), position it freely with transformation tools, then reconnect (N). Beats wrestling a connected crown every time.

Mirror Before You Sculpt

For single anteriors with an intact contralateral, C (copy from scan) then light morphing beats building from a library tooth. Symmetry is the esthetic shortcut.

F6 Rhythm

Confirm with Enter, advance with F6, and keep the right hand on the mouse for the entire design. The techs who look fast aren't clicking Next — they never leave the model.

Cross-Platform Muscle Memory

Running exocad and 3Shape side by side? The traps: pan/zoom modifiers differ (Alt/Shift+RMB here vs. button combos there), redo is Ctrl+Shift+Z here vs. Ctrl+Y there, and single letters mean tools here but visibility groups there.

DME Before Every Migration

Before any Dental System version upgrade, export a full DME from Control Panel. Migrations occasionally eat custom material settings and third-party implant libraries — a two-minute export is the insurance policy.

Version Drift Warning

The official shortcut list has been stable for years, but newer Dental System releases occasionally rebind or add keys. Confirm anything mission-critical against F1 → Appendix B on your installed version.
